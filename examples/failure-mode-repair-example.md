# Failure Mode Repair Example

## Scenario

The first generated headshot looks close, but the eyes feel lifeless and the skin is too smooth.

## Likely Failure

- dead eyes
- plastic skin

## Repair Prompt

`Continue from previous generation, maintaining exact subject identity while adjusting only facial realism. Add sharp catchlights in both eyes so the eyes read alive and bright even at thumbnail size. Preserve visible pores, subtle lines, and natural skin texture. Remove any airbrushed or synthetic smoothing effect.`

## What This Repair Should Improve

- brighter, more believable eyes
- more natural skin texture
- less uncanny polish
- stronger trust signal at LinkedIn size

## Why This Example Matters

A good skill should not just create a first-pass prompt. It should also help recover when the model drifts into common AI portrait errors.
